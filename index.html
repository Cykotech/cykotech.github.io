HTML Reference Sheet

Basic Elements

Tag Name Description
<head> Metadata Header Contains machine-readable information about the document. (i.e. title, scripts, and stylesheets)
<header> Header Typically represents introductory or navigational information.
<body> Body The body of the document.
<h1> Heading Represents a section heading. Has a heirarchy of h1 to h6.
<p> Paragraph Displays a block of text.

Links & Embedding Elements

Tag Name Description
<img> Image Embeds an image in the document using a "src=" attribute.
<a> Anchor Hyperlinks the enclosed text/media using the "href=" attribute.
<video> Video Embeds a video in the document using a "src=" attribute.
